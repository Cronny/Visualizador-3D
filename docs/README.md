# Visualizador 3D

Este proyecto permite cargar y visualizar modelos en formato `.obj` dentro del navegador.

## ¿Qué hace?

- Muestra un modelo 3D a partir de un archivo OBJ.
- Permite mover la cámara en los ejes para cambiar la vista.

## Uso

1. Abre una terminal en la carpeta raíz del proyecto.
2. Ejecuta:

```bash
python3 -m http.server 8000
```

3. Abre en el navegador:

```text
http://localhost:8000
```
